Launcher Folder Structure:

Games have to be in subdirectories of /Games. They can be organised in any directory structure.

Minimal example:

─ /Games/mygame
  ├─ main.py
This type of format will show up with a question mark as the icon and the folder name as the game name.

Minimal example with icon:

─ /Games/mygame
  ├─ main.py
  ├─ icon.bmp
Instead of a question mark, the game will show up with icon.bmp as the icon. For now, the only supported resolution is 64x64.

Full example:

─ /Games/mygame
  ├─ manifest.ini
  ├─ game.py
  ├─ game_icon.bmp
In this example, the names of all the assets are configured in manifest.ini. In the manifest the following lines can be present: 
name = Any Name I Want
version = 0.3
main = game.py
icon = game_icon.bmp
This format is recommended, and at least the version entry should appear in the manifest. The other entries are optional, defaulting to directory name, main.py and icon.bmp.
